These are the codes used to simulate and plot the data in the manusscript: Taking measurement error into account in analyses using categorized exposure by combining MacMahon’s Method and regression calibration, written on behalf of the Measurement Error and Misclassification Topic Group (TG4) of the STRATOS Initiative.

The codes with names starting with "simulation" give the code used for simulation of the data and carrying out the analyses. They are variants on each other
The codes with names starting with "plot" generate the plots in the paper
Not included are the datafiles produced by the simulation and used in the plots, with names starting with "resultssim", as they are too large to upload.
They can be obtained on requestion from the corresponding author
Note that when using the "simulation" codes, the newly generated data will not be exactly the same due to monte-carlo error.
